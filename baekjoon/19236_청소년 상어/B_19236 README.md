## 19236

### B_19236_2.py

이것이 코딩테스트다에 수록된 문제다. 어려운 문제였고 구현과 완전 탐색 유형이다. 구현 해야할 부분들이 그간 풀이한 문제들에 비해 많아서 까다로웠다.

첫 상어 이동 이후, 물고기 이동과 상어 이동을 반복하면서 완전 탐색으로 상어가 먹을 수 있는 물고기 번호의 최대 합을 구해야 한다.

문제 풀이시 유의해야 할 부분들을 주석으로 정리했다.
