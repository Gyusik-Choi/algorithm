# +

## plus operator

문자열 앞에 사용하면 숫자로 변환한다.

```javascript
let stringNumber = '123'
let changeStringNumber = +stringNumber

console.log(changeStringNumber)
console.log(typeof changeStringNumber)
// 123
// number
```
