# 프로그래머스

## 대문자와 소문자

str 클래스의 메소드인 islower 로 소문자 여부를 판단한다. 소문자면 대문자로 변환하기 위해 upper 메소드를 사용하고, 대문자면 소문자로 변환하기 위해 lower 메소드를 사용한다.
